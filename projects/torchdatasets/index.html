torchdatasets

PythonPyTorchData

torchdatasets extends PyTorch's Dataset API with a TensorFlow-data-style pipeline: map transformations applied directly to a dataset and caching of samples in RAM or on disk, so preprocessing is declared once and reused rather than rebuilt each epoch. It supports both map-style Datasets and IterableDatasets and keeps overhead minimal, staying drop-in compatible with existing PyTorch training code. Published on PyPI.
