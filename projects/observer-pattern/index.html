observer-pattern

PythonDesign Patterns

observer-pattern (published as `eventforge`) is a Python library for message-driven task execution: publish/subscribe messaging, local thread/process executors, and RPC over TCP behind a single `@task` decorator that works for both direct and queue-triggered calls.

It ships built-in observers for profiling (timing, metrics, memory, CPU) and request-reply patterns, giving a unified execution and observability layer across local and distributed runs.
