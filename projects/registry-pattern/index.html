registry-pattern

PythonDesign Patterns

registry-pattern is a small Python library providing name-based registries for classes and functions, together with a recursive factory that builds object graphs from JSON-like configuration. Registrations are Pydantic-validated and observable, with built-in meters for performance tracking and reporters for external observability, and the core depends only on the standard library.

It underpins the configuration and instantiation layer of the author's training tooling and is published on PyPI as `registry-pattern`.
