# Images Directory

This directory contains all images for your personal website.

## Structure

```
images/
├── logos/           # Organization/institution logos
│   ├── metu.png
│   ├── eth.png
│   ├── uci.png
│   ├── tubitak.png
│   ├── enocta.png
│   └── ieee.png
├── publications/    # Publication thumbnails/GIFs
│   ├── cinc2021.gif
│   ├── aina2022.gif
│   ├── ieee2022.gif
│   ├── eccv2024.gif
│   ├── dawak2024.gif
│   ├── itu2024.gif
│   ├── dawak2022.gif
│   ├── book2023.gif
│   ├── arxiv2023a.gif
│   └── arxiv2023b.gif

```

## Logos Needed

Add the following organization logos to `images/logos/`:

1. **metu.png** - Middle East Technical University logo
2. **eth.png** - ETH Zürich logo
3. **uci.png** - University of California, Irvine logo
4. **tubitak.png** - TUBITAK logo
5. **enocta.png** - Enocta Inc. logo
6. **ieee.png** - IEEE logo

**Recommended size**: 200x200px, PNG format with transparent background

## Publication GIFs Needed

Add animated GIFs or static images to `images/publications/`:

1. **cinc2021.gif** - CinC 2021 publication
2. **aina2022.gif** - AINA 2022 publication
3. **ieee2022.gif** - IEEE/SRMC 2022 publication
4. **eccv2024.gif** - ECCV 2024 publication
5. **dawak2024.gif** - DaWaK 2024 publication
6. **itu2024.gif** - ITU Journal 2024 publication
7. **dawak2022.gif** - DaWaK 2022 publication
8. **book2023.gif** - Book Chapter 2023
9. **arxiv2023a.gif** - arXiv 2023 (first paper)
10. **arxiv2023b.gif** - arXiv 2023 (second paper)

**Recommended size**: 400x300px (or similar aspect ratio)
**Format**: Animated GIF for visual impact, or PNG/JPG for static images

## Tips

- Use animated GIFs to show your method/results (like Berken Demirel's website)
- Keep file sizes reasonable (< 2MB per GIF)
- Use tools like ezgif.com to optimize GIF sizes
- For logos, prefer PNG with transparent backgrounds
- You can use placeholders initially and replace them later
